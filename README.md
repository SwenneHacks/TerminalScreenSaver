# Bouncy Year
The nostalgic terminal screensaver.

Usage:
```bash
make longest year ever 
```
or change it to this one if you need it (surprise surprise)
```bash
make 2021 work
```
#### how to run it:
This program will run on fullscreen (so make it huge first).
